# 任务 01: 基础框架与配置

## ⚠️ 项目规范提醒
在开始本任务前，请仔细阅读并遵守以下规范（来自 `00-AI生成提示词优化指南.md`）：

### 架构约束
- **严禁**跨层依赖：Domain → App → Infra（只能单向）
- **严禁**在 `internal/domain` 中引入 Gin、GORM 等外部依赖
- **必须**使用 `pkg/autowire` 进行依赖注入
- **必须**在 HTTP Handler 中调用 `handleDomainError()` 映射领域错误

### 错误处理
- **必须**包装错误：`fmt.Errorf("package.Struct.Method: action: %w", err)`
- **必须**在 `internal/domain/errors/` 定义领域错误（哨兵错误）
- **严禁**忽略错误返回值

### 日志规范
- **必须**使用 `pkg/logger` 结构化日志
- **必须**包含字段：`module`, `action`, `success`
- **必须**记录关键业务操作

### 性能约束
- API响应时间 **P99 < 500ms**
- 支持 **100 QPS** 并发
- 数据库查询 **必须**使用 `WithContext(ctx)`
- **严禁**在循环中创建无限制 goroutine

### 代码复杂度
- 函数行数 **≤ 60 行**
- 文件行数 **≤ 1000 行**（硬限制）
- Cyclomatic 复杂度 **≤ 10**

## 任务概述
初始化项目基础框架，实现依赖注入容器、配置管理、日志、响应统一格式等基础设施。

## 上下文依赖
- 无前置依赖（项目起点）

## 涉及文件
- `pkg/autowire/` - 依赖注入框架
- `internal/infra/config/` - 配置加载
- `pkg/logger/` - 结构化日志
- `pkg/response/` - 统一响应格式
- `internal/infra/errors/` - 错误定义

## 详细步骤

### 1.1 依赖注入框架 (pkg/autowire/)
参考现有代码完善或验证：
- [ ] `autowire.go` - 核心DI容器实现
  - 支持 `init()` 注册
  - 支持结构体标签 `autowired:""` 注入
  - 支持接口类型注册和解析
- [ ] 验证DI在 `cmd/server/main.go` 中的集成
- [ ] 添加DI单元测试，覆盖率 > 80%

### 1.2 配置管理 (internal/infra/config/)
- [ ] `config.go` - 配置结构体定义
  - 支持 YAML 文件加载
  - 支持环境变量覆盖
  - 支持 Console 和 Executor 双角色配置
- [ ] 配置项包括：
  - `role`: console|executor
  - `server.port`: HTTP服务端口
  - `database`: PostgreSQL连接配置
  - `etcd`: etcd集群配置
  - `redis`: Redis配置（会话、缓存）
  - `jwt`: JWT认证配置
- [ ] 添加配置加载单元测试

### 1.3 结构化日志 (pkg/logger/)
- [ ] `logger.go` - 基于 `slog` 的封装
  - 支持 JSON 格式输出
  - 支持日志级别（debug/info/warn/error）
  - 支持上下文传递 trace_id
- [ ] 在 Gin 中间件中集成请求日志
- [ ] 添加日志使用示例

### 1.4 统一响应格式 (pkg/response/)
- [ ] 验证现有 `response.go` 实现
- [ ] 确保所有API返回格式：`{ code: number, data: T, msg: string }`
- [ ] code === 0 表示成功，非0表示错误
- [ ] 添加分页响应结构：`{ code, data: { list, total }, msg }`

### 1.5 错误处理封装
- [ ] `internal/infra/errors/domain_errors.go`
  - 定义领域错误（如 ErrNotFound, ErrValidationFailed）
  - 使用标准库 `errors.New` 或 `fmt.Errorf`
- [ ] `internal/ports/http/handler/error_handler.go`
  - 将领域错误映射到HTTP状态码
  - 统一错误响应格式
