### 生成Swagger文档：
- 安装 swag CLI: `go install github.com/swaggo/swag/cmd/swag@latest`
- 执行 `swag init -g cmd/server/main.go -o docs --parseDependency --parseInternal`
- 生成文件：docs/docs.go, docs/swagger.json, docs/swagger.yaml
- 编译验证通过 `go build ./...`

### 注意事项：
- swag init 会完全覆盖 docs/docs.go，不要手写该文件

  - Flow/Resource/Tool/Trigger/Run/Workspace/Environment/Cluster：短路径（依赖 @BasePath /api/v1）
